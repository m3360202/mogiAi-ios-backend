# 自我介绍（SDS法）

## 说明

简洁且不失内容地传达展示自我的信息。为了避免冗长，要聚焦要点进行整理。

## 自我介绍步骤

-   首先报上姓名，其次是大学名称，之后说明"在大学学习什么、目标是什么"，最后提及"校外活动"，并以问候结束。
-   作为大学生，通常隶属于某个学院或系科。无需特别强调学习动机，只需像介绍自己所属专业一样自然地讲述即可。
-   关于学业以外投入的事情，可在制作好"学生经历亮点（ガクチカ）"后，将其概要放入即可。不需要另行构思用于自我介绍的内容。不要为了差异化而说出突兀或夸张的内容。

# 求职"就职轴"（选择企业的核心标准）

## 提问

**"你的求职轴是什么？你基于哪些标准选择企业？"**

## 提问意图

"轴"这个词有时不会被直接使用，但企业想知道的是：\
**你基于什么判断标准选择公司，并希望从这些标准中确认"为什么你会选择我们公司"。**

企业需要能够让他们信服的回答。

------------------------------------------------------------------------

## 面试官关注的点

-   是否能从回答中感受到你的"志愿度"\
-   是否明确地表达"为什么申请本公司"

如果只是一味报考大企业、但没有明确理由，企业就会产生不安：\
"你真的想来我们公司吗？"\
"你对我们公司的工作有热情吗？"

------------------------------------------------------------------------

## 拥有三个求职轴会更顺畅

因为仅靠一个抽象的轴来筛选行业或公司是困难的，因此建议准备多个轴：

示例： - 第一个轴：○○\
- 第二个轴：○○\
- 第三个轴：○○

每个轴都补充一到两句背景说明，回答会更理想。

------------------------------------------------------------------------

# 回答示例：我的求职三大轴

我的求职轴分为三个：

## 1. "从事制造业"

我在大学学习化学的经历，使我对产品开发产生浓厚兴趣，因此希望进入制造型企业工作。

## 2. "不仅进行制造，也追求其后续价值"

我曾参与大学内食品开发，不仅注重美味，也关注价格、食用便捷性等价值。\
这些经验促使我希望创造真正能让用户感到幸福的产品。

## 3. "能最大程度发挥自身优势的环境"

通过参加说明会和OB访谈，我确认能否在企业中发挥自己的优势。

------------------------------------------------------------------------

# 回答要点总结

-   求职轴的提问，是为了确认求职动机的真实性。\
-   要让面试官产生"原来如此，你确实有理由选择我们公司"的信服感。\
-   先说明"轴"，再根据情况补充背景故事会更有效。\
-   如果内容较长，可先说轴，等待面试官询问背景后再补充。
